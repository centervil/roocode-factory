# roocode-factory

Portable AI Armory for Roo Code.

## Structure

- `.roomodes`: Custom mode definitions.
- `.roo/rules/`: Global rules.
- `.roo/rules-{mode}/`: Mode-specific rules.
- `.roo/skills/`: Portable skill packages.

## Modes

- **Meta Maker**: Architect mode for managing this factory.

## Setup

1. Clone this repo.
2. Symlink or copy the contents of `.roo` and `.roomodes` to your project or home directory.
